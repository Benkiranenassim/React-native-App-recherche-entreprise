## Conditions préalables
- JDK 1.8
- Node >= 10
- react-native-cli
- mysql

## Table de la base de donnee
| colonne | type |
| --- | --- |
| id | int) PRIMARY AI |
| name | varchar |
| email | varchar |
| phone | varchar |

## Running Your App
Assurez-vous que votre serveur et mysql fonctionnent déjà, puis lancez l'application
```
$ react-native run-android
or
$ react-native run-ios
```
